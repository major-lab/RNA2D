# RNA 2D utilities

![](doc/figs/logo.jpg)

## Description

Collection of 2D RNA scripts. A bit like Pycogent but with
more exotic stuff.

## Languages used

- python 2/3
- C++
- [julia 0.3](https://github.com/JuliaLang/julia)
